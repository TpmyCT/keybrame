INSTRUCCIONES PARA IMÁGENES
============================

Coloca aquí tus imágenes para usar con el OBS Image Switcher.

Formato recomendado: PNG con fondo transparente
Resolución recomendada: 800x600 o mayor

Cómo agregar imágenes:
----------------------
1. Abre el panel web de administración (http://localhost:5000/admin)
2. Ve a la pestaña "Imágenes"
3. Arrastra tus imágenes o haz click para seleccionar
4. Las imágenes se guardarán automáticamente en esta carpeta

También puedes copiar archivos directamente aquí y se detectarán automáticamente.

Dónde conseguir imágenes:
-------------------------
1. Crear las tuyas propias
2. Buscar "bongo cat sprites" en Google
3. Usar herramientas como Photoshop, GIMP, o Paint.NET
4. Repositorios de sprites gratuitos

Configura qué teclas activan qué imágenes desde el panel web!
